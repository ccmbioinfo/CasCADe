ForCasT

The For Cas Tool, or ForCasT, provides functionality to design, view, and evaluate guides and primers for CRISPR mutagenesis experiments in model organisms. It integrates existing tools including JBrowse, BWA, bedtools, SAMtools, Primer3, BLAST, and Silica to create a complete pipeline.
